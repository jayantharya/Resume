# AI-Powered Professional Resume Portfolio

## Overview
A dynamic, AI-driven portfolio platform designed to showcase professional experience and technical project documentation.

## Features
* **AI Integration:** Leverages generative AI for smart content analysis.
* **Modular Architecture:** Built with a clear separation between frontend and backend.
* **Responsive UI:** Fully optimized for mobile and desktop viewing.

## Tech Stack
* **Frontend:** React/Next.js
* **Backend:** Python
* **Deployment:** Vercel

## How to Run
1. Clone the repository: `git clone https://github.com/your-username/Resume.git`
2. Install dependencies:
   - Frontend: `cd frontend && npm install`
   - Backend: `pip install fastapi uvicorn pydantic google-genai`
3. Run the project: `npm run dev`
